Aplicația folosește AI de la Anthropic. Ai nevoie de o cheie API gratuită.

1
Creează un cont Anthropic
Mergi pe console.anthropic.com și creează un cont gratuit. Vei primi credite pentru a testa aplicația.
2
Generează o cheie API
În consolă, mergi la API Keys → Create Key. Copiază cheia (începe cu sk-ant-...)
3
Introdu cheia mai jos
Cheia se salvează local pe dispozitivul tău. Nu se trimite nicăieri altundeva.
🔒 Cheia se salvează doar local pe telefon (localStorage).
Nu o trimitem pe niciun server propriu.
